# VideoFileRenamer

[English README](README.md)

`VideoFileRenamer` 是一个 Windows 桌面工具，用于批量整理视频文件名。它会从现有文件名中识别日期和时间，先展示重命名预览，确认后再执行修改。

## ✨ 功能

- 选择素材文件夹并扫描文件。
- 可选择整理后目录，用于存放改名后的文件。
- 可选择只处理常见视频格式。
- 可选择包含子文件夹。
- 可选择改名后移动文件或复制文件。
- 将识别到的日期和时间转换为统一命名格式。
- 支持 NVIDIA Replay 风格文件名。
- 执行前预览原文件名、新文件名、状态和所在文件夹。
- 将改名后的文件移动或复制到指定整理后目录。
- 每次批量处理都会生成 CSV 操作日志。
- 目标文件名冲突时自动追加编号，避免覆盖文件。

## 🏷️ 命名规则

识别到日期和时间时：

```text
yyyy-MM-dd_HH-mm-ss.ext
```

只识别到日期时：

```text
yyyy-MM-dd.ext
```

如果重命名位置中已存在同名目标文件，会自动追加编号：

```text
2026-05-31_13-20-05_02.mp4
```

## 🎬 支持的视频扩展名

启用“只处理常见视频格式”时，默认处理以下扩展名：

```text
.mp4 .mov .mkv .avi .wmv .flv .webm .m4v .mts .m2ts .ts
```

## 🖱️ 使用方式

1. 启动程序。
2. 选择素材文件夹。
3. 如果需要把改名后的文件移动到指定目录，选择整理后目录。
4. 选择移动或复制模式。
5. 按需启用或关闭“只处理常见视频格式”和“包含子文件夹”。
6. 点击预览，检查重命名结果。
7. 确认无误后执行重命名。

## ⚠️ 注意事项

- 程序会先预览再改名，但最终重命名操作会直接修改磁盘上的文件名。
- 移动模式会修改原文件。
- 复制模式会保留原文件，并生成改名后的副本。
- 如果选择了整理后目录，文件会被改名并移动或复制到该目录。
- 如果没有选择整理后目录，改名后的文件会写入原文件夹。
- CSV 日志会写入可执行文件同级的 `logs` 文件夹。
- 日志只保留最新 10 份。
- 建议先用少量测试文件验证命名效果。
- 文件名中没有可识别日期时会被跳过。
